# S3 Private Bucket Security

## Objective
Create a secure S3 bucket and prevent public access.

## Services Used
- Amazon S3

## Steps Performed
1. Created a new S3 bucket
2. Enabled Block Public Access
3. Uploaded a file
4. Tested access via URL

## Security Configuration
- Blocked all public access
- Ensured bucket is private

## Result
- Access to file returned "Access Denied"

## What I Learned
- Importance of securing S3 buckets
- Common misconfigurations that lead to data leaks
